Register your website with Apple Pay

Get your website ready to accept Apple Pay in three steps. If you work with a developer, they can help with step three.

1. Add your website

We'll register this site with Apple Pay. Be sure to use your fully qualified domain. Example: domain.com

2. Get your domain association file

This file acts as proof you own the website. You can download it below.

↓ apple-developer-merchantid-domain-association

3. Host your domain association file

This exact address is where Apple will check for the file. A developer can take care of this step for you using our developer documentation. Once the file's in place, you'll select Register Domain.
